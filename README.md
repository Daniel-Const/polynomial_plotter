# Polynomial Plotter

Simple plotting app for polynomials written in C++ with OpenGL / GLUT

## Setup

1. Clone repository

   ```bash
   > git clone https://github.com/Daniel-Const/poynomial_plotter.git
   > cd polynomial_plotter
   ```

2. Build files

   ```bash
   > mkdir bin
   > cd bin
   > cmake ..
   > make
   ```

3. Run

   ```bash
   ./polynomial
   ```
